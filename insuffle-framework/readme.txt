== Description ==

Insuffle Framework est un thème WordPress moderne, performant et entièrement personnalisable. Conçu pour être réutilisable sur tous vos projets, il offre une base solide et élégante pour créer des sites web professionnels.

= Caractéristiques principales =

* Design épuré et moderne
* 100% Responsive (mobile-first)
* Compatible Gutenberg
* SEO optimisé (Schema.org, Open Graph)
* Performances optimisées
* 5 palettes de couleurs + mode personnalisé
* Typographie exclusive Poppins
* Customizer complet
* 6 templates personnalisés
* Blocs et patterns Gutenberg stylisés
* Accessible (WCAG 2.1 AA)

= Templates disponibles =

* Page standard
* Article de blog
* Formation
* Offre / Service
* Contact
* Landing Page
* Page 404 personnalisée

= Personnalisation =

Le thème offre de nombreuses options de personnalisation via le Customizer WordPress :

* Couleurs (palette complète personnalisable)
* Typographie (taille, graisse, hauteur de ligne)
* En-tête (layout, sticky, bouton CTA)
* Footer (colonnes, texte, branding)
* Mise en page (largeur, espacements, arrondis)
* Blog (disposition, affichage)

== Installation ==

1. Dans votre tableau de bord WordPress, allez dans Apparence > Thèmes > Ajouter
2. Cliquez sur "Téléverser un thème"
3. Sélectionnez le fichier insuffle-framework.zip
4. Cliquez sur "Installer maintenant"
5. Activez le thème

Ou manuellement :

1. Téléchargez le fichier ZIP du thème
2. Décompressez-le
3. Uploadez le dossier dans /wp-content/themes/
4. Activez le thème dans Apparence > Thèmes

== Configuration ==

= Après l'activation =

1. Allez dans Apparence > Personnaliser
2. Configurez votre logo et favicon (Identité du site)
3. Choisissez votre palette de couleurs (Couleurs Insuffle)
4. Créez vos menus (Apparence > Menus)
5. Configurez les zones de widgets (Apparence > Widgets)

= Menus recommandés =

* Menu Principal (emplacement header)
* Menu Footer (emplacement footer)
* Liens Sociaux (emplacement social)

= Widgets =

Le thème offre 5 zones de widgets :

* Sidebar (barre latérale)
* Footer 1, 2, 3, 4 (colonnes du footer)

== Frequently Asked Questions ==

= Le thème est-il gratuit ? =

Oui, Insuffle Framework est gratuit et distribué sous licence GPL v2 or later.

= Le thème est-il compatible avec Gutenberg ? =

Oui, le thème est entièrement compatible avec l'éditeur Gutenberg et inclut des blocs stylisés et des patterns personnalisés.

= Puis-je modifier les couleurs ? =

Oui, vous pouvez choisir parmi 5 palettes pré-configurées ou créer votre propre palette via le Customizer.

= Le thème est-il compatible avec les plugins ? =

Oui, le thème est compatible avec la plupart des plugins WordPress populaires :
* Yoast SEO / Rank Math
* Contact Form 7 / Gravity Forms
* WP Super Cache / WP Rocket
* Smush / ShortPixel

= Le thème est-il responsive ? =

Oui, le thème est 100% responsive et optimisé pour tous les écrans (mobile, tablette, desktop).

= Comment créer une landing page ? =

1. Créez une nouvelle page
2. Dans "Attributs de page", sélectionnez le template "Landing Page"
3. Publiez votre page

= Le thème est-il optimisé pour le SEO ? =

Oui, le thème intègre nativement :
* Balises sémantiques HTML5
* Schema.org / JSON-LD
* Open Graph (Facebook, LinkedIn)
* Twitter Cards
* Fil d'Ariane
* Métadonnées complètes
